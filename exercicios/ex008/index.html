negrito / destaque

italico / enfase

nessa frase tenho um termo em italico, usando a tag I (nao semantica)

nessa frase temos um termo em enfase, usando a tag Em (semantica)

texto marcado

é possivel criar um texto marcado usando a tag Mark (nesse texto eu usei style para mudar a cor do marcador)

nesse aqui eu não mudei a cor do marcador (mas mudei utilizando o parametro style geral da pagina)

texto grande e pequeno

estou criando um texto grande e um texto pequeno no mesmo paragrafo, usando as tags Big e Small

texto deletado

posso marcar um texto como excluido, para indicar que deve ser lido mas nao considerado, usando a tag Del

texto inserido

a tag U tambem pode ser usada para isso mas ela é não semantica

texto sobrescrito

para inserir coisas como x20+3, usando a tag Sup

texto subscrito

para inserir coisas como h2O, usando a tag Sub
